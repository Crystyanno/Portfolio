MINHAS HABILIDADES.

Html

(Linguagem de Marcação de Hipertexto), HTML é a base da web, usando tags para organizar e descrever conteúdo, permitindo formatação, inserção de mídia e criação de links. É interpretado pelos navegadores para mostrar páginas web aos usuários.

Css

As Cascading Style Sheets (CSS) são usadas para adicionar estilos às páginas da web, podendo ser aplicadas diretamente nas tags HTML ou em um arquivo separado. Isso permite controlar a aparência e o layout dos elementos HTML.

JavaScript

O JavaScript é uma linguagem de programação interpretada, de alto nível e multiparadigma, frequentemente utilizada junto com HTML e CSS para criar páginas web. Possui tipagem dinâmica fraca e é uma das tecnologias fundamentais da World Wide Web.
